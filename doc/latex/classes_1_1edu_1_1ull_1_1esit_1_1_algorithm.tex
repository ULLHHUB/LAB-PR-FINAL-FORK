\doxysubsection{Detailed Description}
Context class for pathfinding algorithms using the Strategy pattern. Supports DFS, BFS, A\texorpdfstring{$\ast$}{*}, Dijkstra, Greedy Best-\/\+First Search, and Bidirectional Search. Each algorithm is implemented as a separate strategy class.
